Geoclipse

Add free (libre) maps to your Eclipse RCP project.

Geoclipse is an Eclipse bundle, that allows you to add mapping capabilities to your Eclipse Rich Client applications. The API tries to mimic that of Google Maps on Android. The maps used are from the OpenStreetMap project, however. But Geoclipse is extensible in that regard.

Current State

Geoclipse is in a fairly good shape at the moment. It is usable and you can create working applications with it. Some features are still missing though (see ToDo below). The documentation is good, but not complete yet, I am working on this at the moment.

There is a sample application, called AndroidLoc, that visualizes Android location cache files.

If you want to work on anything, feel free to contact me. If you have any patches, you would like to see included, send me a pull request right here on GitHub.

You should use this version right here for new development. There is an older version of Geoclipse at SourceForge. I consider it deprecated, but will leave it there for now.
